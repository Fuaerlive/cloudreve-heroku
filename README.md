# cloudreve Heroku

## 概述

用于在 Heroku 上部署 [cloudreve](https://cloudreve.org/)。

**部署前请三思。**

## 镜像

本镜像基于alpine:3.9.5

cloudreve二进制可执行文件在此镜像下编译生成，可能不适用于其他镜像

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)
